# Changelog

All notable changes to Driftmark are documented here.

## [1.0.0] - 2026-03-25

### Added
- Interactive world map with country drill-down to city pins
- City-level ratings across 10 categories (safety, food, transport, friendliness, affordability, cleanliness, nightlife, natural beauty, Wi-Fi, overall experience)
- Rating radar chart and bar chart visualizations
- Group travel: create squads of up to 4, share maps, compare ratings
- Real-time group sync via Supabase Realtime
- Travel timeline with photos and reviews
- Trip planning with dates and budget tracking
- 7 achievement badges (first stamp, continental, globe trotter, critic, squad goals, home away, city explorer)
- Offline-first with queue-based sync
- Google OAuth authentication
- Photo upload with automatic EXIF stripping (privacy protection)
- 1,071 pre-seeded cities across 195 countries
- Cross-platform: iOS, Android, and Web
- OTA updates via EAS Update
- Full CI/CD pipeline with GitHub Actions
